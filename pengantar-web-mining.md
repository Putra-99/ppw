# Pengantar Web Mining

## 1. Definisi & Tujuan

- **Web Mining** = proses menemukan & mengekstrak informasi dari internet dengan teknik data mining.
- Tujuan:
  - Memahami & menggunakan konsep dasar web mining.
  - Melakukan penelitian sistem informasi dalam konteks web mining.

---

## 2. Tantangan Web Mining

- Web sangat **besar**, **kompleks**, **dinamis**, dan tidak spesifik domain.
- Data dalam berbagai format: HTML, XML, teks.
- Dibutuhkan teknik preprocessing yang baik.

---

## 3. Taxonomy Web Mining

1. **Web Content Mining**

   - Fokus pada konten halaman web: teks, gambar, audio, video, tabel.
   - Aplikasi:
     - Text Mining → ekstraksi informasi, klasifikasi dokumen, clustering, sentiment analysis.
     - Image/Audio/Video Mining.

2. **Web Usage Mining**

   - Analisis data penggunaan web (log server, cookie, clickstream).
   - Aplikasi:
     - Rekomendasi produk.
     - Personalized search.

3. **Web Structure Mining**
   - Analisis struktur hyperlink & jaringan sosial.
   - Aplikasi:
     - PageRank (Google).
     - Analisis influencer & komunitas.

---

## 4. Proses Web Mining

1. **Gathering & Exploration**
   - Crawling data, API, eksplorasi data.
2. **Preprocessing & Transformation**
   - Pembersihan, reduksi dimensi, seleksi fitur, embedding teks.
   - 70-80% waktu proyek ada di tahap ini.
3. **Actual Data Mining**
   - Menerapkan algoritma (klasifikasi, clustering, asosiasi).
   - Evaluasi model → iterasi & perbaikan.

---

## 5. Contoh Tugas Text Mining

- **Klasifikasi teks** → (Naive Bayes, SVM, Deep Neural Net, Transformers).
- **Clustering dokumen** → (K-Means, Hierarchical, S-BERT).
- **Sentiment analysis** → positif/negatif/netral.
- **Information Extraction** → mengubah data semi-terstruktur (HTML, API, tabel) menjadi data terstruktur.

---

## 6. Aplikasi Nyata

- Google Search → PageRank.
- Twitter/Instagram → deteksi influencer.
- Sistem rekomendasi → berdasarkan penggunaan & komunitas.
- Analisis clickstream → memahami perilaku pengguna.

---

## 7. Bidang Ilmu Terkait

- Natural Language Processing (NLP)
- Machine Learning
- Database Systems
- Social Network Analysis
- Graph Mining
